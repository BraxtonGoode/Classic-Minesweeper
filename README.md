# Overview

This game is built upon the classic game of Minesweeper, where the player must uncover all the safe tiles without detonating any mines. The game board consists of a grid of tiles, some of which contain hidden mines. The player's objective is to click on the tiles to reveal what lies beneath them. If a tile contains a mine, the game is over. If it does not contain a mine, it will display a number indicating how many adjacent tiles contain mines. You can also place red flags with the use of the right click to mark suspected mines. The player can use this information to deduce which tiles are safe to click on and which ones may contain mines. The game continues until the player successfully uncovers all the safe tiles or clicks on a mine.

The purpose of writing this software was to create a fun and engaging game that challenges players' problem-solving skills and strategic thinking. I wanted to recreate the classic Minesweeper game while also adding my own unique features and improvements. Additionally, I aimed to enhance my programming skills and gain experience in game development using Python and the Arcade library.

# Development Environment

The game was developed using Python 3.10 and the Arcade library for graphics and game development. I used Visual Studio Code as my code editor, which provided helpful features such as syntax highlighting and debugging tools. Additionally, I utilized Git for version control to manage changes to the codebase throughout the development process.

The game was written in Python, a versatile and widely-used programming language known for its readability and ease of use. The Arcade library was used to create the game's graphics and handle user input, making it easier to develop a visually appealing and interactive game. The combination of Python and Arcade allowed for efficient development while also providing a solid foundation for learning game development concepts.
